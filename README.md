# CodeLeap Network

This project is a full-stack application developed for the CodeLeap challenge. It consists of a Django backend and a React frontend.

## Documentation

For detailed information on each part of the project, please refer to the specific README files:

- [Backend Documentation](./backend/README.md)
- [Frontend Documentation](./frontend/README.md)

## What is implemented?

- Simple API in Django
- React frontend based on [Figma design](https://www.figma.com/design/0OQWLQmU14SF2cDhHPJ2sx/CodeLeap-Engineering-Test?node-id=29005-174&t=Wnog5ZpVKz3tGjvX-0)
- Animations
- Login and Signup
- Create, Read, Update and Delete posts
- Custom Hooks
- Pagination with Infinite Scroll
- Responsive design
